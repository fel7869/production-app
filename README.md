# 产量统计助手

一个简单实用的产量录入与统计工具，专为生产加工场景设计。

## ✨ 功能特性

### 📝 核心功能
- **每日录入** - 记录型号、加工类型、数量、单价
- **自动计算** - 实时计算小计金额
- **数据保存** - 本地存储，关闭浏览器也不会丢失

### 📊 统计分析
- **今日合计** - 当天所有记录的总金额
- **累计总计** - 所有历史记录的总金额
- **月份统计** - 选择任意月份查看该月合计
- **智能筛选** - 按型号、加工类型筛选记录

### 📱 PWA 支持
- 添加到手机主屏幕，像原生 APP 一样使用
- 离线可用（首次加载后）
- 流畅的操作体验

## 🚀 使用方法

### 方法1：直接打开
双击 `index.html` 文件，在浏览器中直接使用。

### 方法2：手机使用
1. 将整个 `production-app` 文件夹传到手机
2. 用手机浏览器打开 `index.html`
3. 点击浏览器菜单 → "添加到主屏幕"

### 方法3：局域网访问（推荐）
1. 双击 `启动服务器.bat`
2. 查看显示的局域网 IP 地址
3. 在手机浏览器输入：`http://[IP地址]:8000`
4. 添加到主屏幕

## 💾 数据存储

所有数据保存在浏览器的 localStorage 中：
- 数据仅存储在本地设备
- 不会上传到任何服务器
- 清除浏览器数据会删除记录

## 📋 使用说明

1. **录入数据**
   - 选择日期（默认今天）
   - 输入型号、加工类型、数量、单价
   - 点击"保存记录"

2. **查看统计**
   - 点击底部"统计"标签
   - 查看今日合计、累计总计
   - 选择月份查看月度统计

3. **筛选记录**
   - 点击底部"记录"标签
   - 使用下拉菜单按型号或类型筛选
   - 点击"×"可删除记录
